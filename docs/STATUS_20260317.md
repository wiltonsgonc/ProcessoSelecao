- **Sintoma**: `Http failure response for http://localhost:5002/api/processosselecao: 0 Unknown Error`
- **Causa**: Backend travado devido ao `dotnet watch`.
- **Solução**: Backend agora usa `Dockerfile.prod` (sem `dotnet watch`).
- **Verificação**: 
  ```bash
  curl http://localhost:5002/api/processosselecao
  ```

  ```bash
  curl -X POST http://localhost:5002/api/processosselecao -d '{"nome":"Teste","vagasDisponiveis":5}'
  # Retorna ID sequencial (1, 2, 3, 4...)
  ```

### 3. Debugs removidos
- **Backend**: Arquivo temporário `ResetIdentity.cs` removido.
