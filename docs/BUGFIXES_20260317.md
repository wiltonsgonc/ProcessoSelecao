# Correções de Bugs - 2026-03-17

## Problema 1: Erro ao criar processo
- **Erro**: `Http failure response for http://localhost:5002/api/processosselecao: 0 Unknown Error`
- **Causa**: O backend estava travado devido ao `dotnet watch` e não respondia às requisições.
- **Solução**:
  1. Modificado `docker-compose.yml` para usar `Dockerfile.prod` em vez de `Dockerfile.dev`.
  2. `Dockerfile.prod` executa `dotnet run` diretamente sem `dotnet watch`.
  3. Reconstruído o backend com o novo Dockerfile.
  4. O backend agora responde corretamente na porta 5002.

## Problema 2: IDs sequenciais começando em valores altos
- **Solução**: Resetado o `IDENTITY` das tabelas usando `DBCC CHECKIDENT`. Os IDs agora começam em 1.

## Problema 3: Remoção de debugs

- **Frontend**: Comentados todos os `console.log` nos arquivos:
